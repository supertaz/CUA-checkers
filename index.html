AGENT API

Coords: files a..h left→right, ranks 1..8 bottom→top.
Red starts at ranks 1–3, Black at 6–8. Red moves up, Black moves down.

DOM: each playable square has id="sq-filerank", data-square, data-piece, aria-label.

JS API on window.checkers:
  • getState()
  • getLegalMoves()
  • move("a3","b4")
  • moveSan("a3-b4") or "a3xc5"
  • reset(), undo()
  • onChange(fn)
